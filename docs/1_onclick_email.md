# Initial Setup:
+ Visit the official page of n8n : https://www.n8n.io.
+ First create an n8n account if you don't have, if you have one signin/login to your account.
+ Next create an empty workflow in your personal folder.
      
# Steps to Create an ONCLICK EMAIL WORKFLOW:
+ First create an "ON CHAT MESSAGE TRIGGER" by clicking "Add first step.." icon.
+ Next click the "+" symbol and create an Manual Trigger. 
+ Then add an Email Node to the Manual Trigger by clicking the "+" icon near the Manual Trigger.
+ Click the Email Node and Create a New Credential.
+ Signin with your Gmail account and fill in the Recipient Email ID, Subject, Message.
+ Now click the "Execute Workflow" button and check if the mail has reached the recipient email.
+ Now you are readty to implement your onclick email workflow üëçüëç.
